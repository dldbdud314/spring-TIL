## 서블릿
### 서블릿이란?
**request 읽어 들이고 response 생성**해주는 역할을 한다! 개발자는 비즈니스 로직 구현만 신경 쓰면 됨
#### 요청
> 1. GET 쿼리 파라메터
> 2. POST HTML form
> 3. HTTP message body에 직접 데이터 담는 방식 (주로 JSON)
